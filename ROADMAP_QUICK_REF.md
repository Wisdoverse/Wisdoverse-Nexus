## 立即行动清单

### 本周必须完成

1. **修复 JWT 硬编码**（1-2 天）
   ```rust
   // 移除默认值，强制要求环境变量
   let secret = std::env::var("JWT_SECRET")
       .expect("JWT_SECRET must be set");
   ```

2. **开始团队招聘**
   - [ ] 3 名 Rust 后端工程师
   - [ ] 1 名 DevOps
   - [ ] 1 名 DBA

3. **技术决策确认**
   - [ ] CRDT: 自研 vs 集成 yrs
   - [ ] SFU: 自研 vs 集成 LiveKit
   - [ ] 状态存储: Redis vs PostgreSQL

### 本月必须完成

1. **添加数据库 RLS**
   ```sql
   ALTER TABLE rooms ENABLE ROW LEVEL SECURITY;
   CREATE POLICY tenant_isolation ON rooms
     USING (tenant_id = current_setting('app.current_tenant')::uuid);
   ```

2. **开始网关无状态化设计**
   - 设计 Redis 数据结构
   - 迁移 WebSocket 状态

3. **开始替换 unwrap**
   - 使用 `cargo clippy` 检测
   - 逐模块替换为 `?` 或 `expect`
