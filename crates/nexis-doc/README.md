# nexis-doc

Real-time document collaboration domain crate for document metadata, snapshots, comments, and CRDT operation modeling.

## Modules

- `document`: `Document`, `DocMetadata`, `DocVersion`
- `snapshot`: `DocSnapshot`, `SnapshotMeta`
- `comment`: `Comment`, `CommentThread`, `CommentAnchor`
- `crdt`: `CRDTOperation`, `OpType`, `OperationLog`
- `error`: `DocError`, `DocResult`
