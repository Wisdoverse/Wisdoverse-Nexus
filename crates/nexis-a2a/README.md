# nexis-a2a

Agent-to-Agent (A2A) protocol primitives for Nexis.

## Scope

- Agent identity and capability models
- Discovery filters and in-memory registry
- Communication envelopes and payloads
- Collaboration task and handoff types
- Shared crate error/result types

## Status

This crate currently provides foundational data types with minimal behavior, intended for integration and protocol evolution in later phases.
