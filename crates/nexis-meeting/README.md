# nexis-meeting

Meeting collaboration domain crate for rooms, participants, WebRTC signaling, recording metadata, and AI-generated summaries.

## Modules

- `room`: `MeetingRoom`, `RoomConfig`, `RoomState`
- `participant`: `Participant`, `ParticipantRole`, `MediaState`
- `signaling`: `SignalMessage`, `SignalType`
- `recording`: `Recording`, `RecordingState`, `TrackType`
- `summary`: `MeetingSummary`, `ActionItem`
- `error`: `MeetingError`, `MeetingResult`
