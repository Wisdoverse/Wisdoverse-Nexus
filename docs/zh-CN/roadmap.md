# 路线图

Wisdoverse Nexus 仍处于 pre-1.0 阶段。路线图表示当前方向，不代表
固定交付日期、商业承诺或兼容性保证。

## 当前重点

- 稳定 `nexis-gateway` 的 HTTP/WebSocket API。
- 继续收敛 `nexis-protocol`、`nexis-context`、`nexis-memory` 等 crate 的公共边界。
- 保持 TypeScript SDK、移动端和 Web 示例可构建。
- 让文档、CI、发布流程适合 GitHub 源码可用协作。

## 近期方向

- 插件运行时和示例文档。
- MCP 与 A2A 互操作。
- 多租户边界、认证和审计日志增强。
- 可复现 benchmark 与部署容量说明。
- 更多本地开发和贡献者验证指南。

## 长期方向

- 更完整的 AI 协作协议和身份模型。
- 更强的多模型提供商抽象。
- 联邦部署和混合部署能力。
- 社区插件、SDK 和集成示例。

## 贡献

路线图会随代码和社区反馈调整。功能建议和问题反馈请使用：

- [GitHub Issues](https://github.com/Wisdoverse/Wisdoverse-Nexus/issues)
- [GitHub Discussions](https://github.com/Wisdoverse/Wisdoverse-Nexus/discussions)
