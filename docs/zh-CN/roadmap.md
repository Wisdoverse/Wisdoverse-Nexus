# 路线图

Nexis 的发展规划和里程碑。

## v0.1 — 基础架构 ✅

**状态**: 已完成

- [x] WebSocket 网关 (nexis-gateway)
- [x] 核心类型系统 (nexis-core)
- [x] 消息协议 (nexis-protocol)
- [x] 上下文管理 (nexis-context)
- [x] 10 万+ 并发连接支持
- [x] JWT 认证
- [x] Prometheus 指标
- [x] CI/CD 流程

## v0.2 — 插件系统 🚧

**状态**: 进行中

- [ ] Wasm 插件运行时 (nexis-plugin)
- [ ] 插件生命周期钩子
- [ ] 插件 API 稳定化
- [ ] 插件示例和文档
- [ ] 插件市场 (Beta)

## v0.3 — 企业功能

**计划**: 2026 Q2

- [ ] SSO / SAML 集成
- [ ] RBAC 权限系统
- [ ] 审计日志
- [ ] 数据加密 (静态 + 传输)
- [ ] 多区域部署支持

## v0.4 — AI 增强

**计划**: 2026 Q3

- [ ] 内置 AI 摘要
- [ ] 情绪分析
- [ ] 智能消息路由
- [ ] 多模型提供商支持
- [ ] AI 代理身份管理

## v0.5 — 生态扩展

**计划**: 2026 Q4

- [ ] MCP (Model Context Protocol) 集成
- [ ] A2A (Agent-to-Agent) 通信
- [ ] 联邦协议 (nexis-federation)
- [ ] 移动 SDK (iOS/Android)
- [ ] CLI 工具增强

## 长期愿景

- **插件市场** — 类似 VS Code 的插件生态
- **行业解决方案** — 金融、医疗、教育等行业模板
- **混合部署** — 云端 + 边缘 + 本地
- **AI 原生团队** — 人与 AI 成员无缝协作
