# 安全概览

Wisdoverse Nexus 的安全文档聚焦当前仓库中可验证的控制面，而不是历史审计
快照或第三方认证声明。

## 报告漏洞

请不要通过公开 issue 或 PR 报告安全漏洞。请使用 GitHub Private Security
Advisory：

- <https://github.com/Wisdoverse/Wisdoverse-Nexus/security/advisories/new>

## 安全文档

英文文档是安全控制说明的主版本：

- [Baseline Profile](/en/architecture/security/baseline-profile)
- [Enterprise Profile](/en/architecture/security/enterprise-profile)
- [Key Management](/en/architecture/security/key-management)

这些页面说明仓库控制、运行时加固、密钥管理和由部署方负责的合规治理。它们不
构成第三方审计报告、认证报告或合规证明。
