# Security Architecture Overview

Wisdoverse Nexus security documentation is split into layered profiles:

- [Baseline Profile](baseline-profile.md)
- [Enterprise Profile](enterprise-profile.md)
- [Key Management](key-management.md)

These documents cover repository controls, runtime hardening, secret management,
and operator-owned compliance governance. They are maintained by the project and
are not third-party audit reports, certification reports, or compliance
attestations.
