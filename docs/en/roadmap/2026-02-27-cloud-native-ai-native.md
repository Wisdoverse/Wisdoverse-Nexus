# Planning Note: Cloud Native and AI Native Direction

This historical planning note records product direction. It is not a release
commitment or production benchmark.

Target areas:

- Kubernetes-oriented deployment assets.
- Real-time gateway reliability and observability.
- AI provider integration as an optional collaboration capability.
- Clear separation between stable APIs and experimental modules.

Before any production-scale claim is published, the project should include:

- Reproducible benchmark commands and raw results.
- Hardware and runtime environment details.
- Deployment topology and configuration.
- Monitoring evidence from the target environment.
- Clear release or commit SHA references.
