# Jira Plugin for Wisdoverse Nexus

Integration with Atlassian Jira for issue tracking.

## Features

- `/jira create <summary>` — create an issue
- `/jira list` — list open issues
- `/jira sprint` — show current sprint status

## Configuration

Set `JIRA_BASE_URL`, `JIRA_EMAIL`, `JIRA_API_TOKEN` environment variables.
